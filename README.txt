Luxe Orbit Browser - edición visual con base funcional

Contenido:
- index.html
- styles.css
- app.js
- _headers

Qué es esta versión:
- mantiene la base funcional del paquete más estable
- recupera una apariencia más cercana al diseño premium anterior
- lista para subir a Netlify

Cómo subirlo:
1. Descomprime el ZIP
2. Sube la carpeta completa a Netlify o su contenido
3. El archivo principal es index.html

Notas:
- la cuenta y los ajustes se guardan de forma local en el navegador/dispositivo
- el historial de sesión no debe conservarse como historial permanente de cuenta
- algunas páginas externas pueden bloquear el visor interno por seguridad propia
